# AntiSpam Moriarty Bot — v2

Антиспам-бот для Telegram-групп + **автоответы по ключевым словам** + **панель управления в ЛС только для владельца**.

## Что нового в этой версии

- 💬 **Автоответы.** Пишешь в чат слово — бот отвечает заданным текстом. Слов можно сколько угодно. Пример: `банан` → `300 руб`.
- 🛠 **Панель управления в ЛС** (на кнопках, как в примере) — управляешь только ты.
- ⌨️ **Команды для удалённого управления** прямо из ЛС с ботом.
- 💾 Все изменения сохраняются и применяются на лету, перезапуск кода не нужен.

Старые функции на месте: блокировка invite-ссылок, сокращателей и спам-доменов, фильтр стоп-слов, антифлуд (мут), бан при входе по спам-имени.

## Деплой (как и раньше)

1. Залей все файлы в репозиторий на GitHub: `bot.py`, `requirements.txt`, `Procfile`, `railway.toml` (`.env.example` и `.gitignore` — по желанию).
2. На **railway.app** → New Project → Deploy from GitHub repo → выбери репозиторий.
3. **Variables → New Variable:**
   - `BOT_TOKEN` = токен от @BotFather
   - `ADMIN_IDS` = `8387802287` (уже стоит по умолчанию в коде, но переменной надёжнее)
4. Deploy. В группе дай боту права администратора: **удалять сообщения, блокировать и ограничивать участников.**

## Управление ботом (только владелец)

Напиши боту в личку `/start` или `/panel` — откроется меню с кнопками:

- **📊 Статус** — что включено сейчас
- **🔘 Функции** — включить/выключить любой модуль одной кнопкой
- **⚙️ Антифлуд** — лимит / период / длительность мута кнопками ➖ ➕
- **💬 Ключевые слова** — добавить/удалить автоответы, переключить режим
- **🚫 Стоп-слова** и **🔗 Спам-ссылки** — списки с кнопкой ❌ для удаления

### Команды (альтернатива кнопкам)

```
/panel              открыть панель
/status             текущие настройки
/id                 узнать свой ID и ID чата

/add банан = 300 руб   добавить автоответ
/del банан             удалить автоответ
/list                  список автоответов

/addword казино     добавить стоп-слово
/delword казино     удалить стоп-слово
/words              список стоп-слов

/addlink spam.ru    добавить спам-домен
/dellink spam.ru    удалить спам-домен
/links              список доменов
```

### Режимы срабатывания автоответов

- **целое слово** (по умолчанию): `банан` сработает на «банан», но не на «бананы».
- **любое вхождение**: `банан` сработает и на «бананы», и внутри других слов.

Переключается кнопкой 🔁 в разделе «Ключевые слова».

## ⚠️ Сохранение настроек после передеплоя

Файловая система Railway сбрасывается при пересборке. Чтобы автоответы и списки **не слетали** после каждого передеплоя:

1. В проекте Railway добавь **Volume** и смонтируй его в путь `/data`.
2. Добавь переменную `DATA_DIR=/data`.

Без Volume настройки переживают перезапуски, но обнуляются при пересборке (новый коммит и т.п.).

## Безопасность

- Управлять ботом может только ID из `ADMIN_IDS`. Любой другой человек в ЛС получит вежливый отказ.
- Токен держи только в переменных Railway, никогда не коммить его в код.
